## アプリ名
game_review_app
## サイト概要
📝 サイトテーマ

ゲームの評価や感想を書き、様々な人に見てもらうレビューサイト。

💡 テーマを選んだ理由

私は小さいころからゲームに親しみ、これまでさまざまなタイトルを遊んできた経験があります。
しかし、ゲームの魅力や実際のプレイ感を知るための情報は散在しており、初心者や興味を持った人が作品を選びづらいという課題があります。

また、ユーザー目線で分かりやすくまとめられたレビューを求めている人も多く、信頼できる感想や評価へのニーズは高いと感じています。

こうした課題を解決するため、ゲームの魅力を整理して伝えられるレビューサイトを制作しました。

🎯 ターゲットユーザ

・対象のゲームを買うのに迷っている人

・プレイ時間の目安や満足度を知りたい人

・自分に合ったジャンルのゲームを探したい人

⏰ 主な利用シーン

・ゲームの難易度を知りたい時

・ゲームの簡単な物語の内容を知りたい時

・ゲームの想定プレイ時間を知りたい時

✨ 実装済み機能一覧
▼ ユーザー関連

・ユーザー登録 / ログイン（Devise）

・プロフィール編集

・マイページ表示

▼ 投稿（レビュー）機能

・レビュー投稿

・レビュー編集 / 削除（本人のみ）

・コメント機能

・いいね機能

・タグ付け

・ジャンル（グループ）分類

・投稿詳細表示

▼ 一覧表示・検索

・新着投稿一覧

・タグ別一覧

・ジャンル別一覧

・キーワード検索

▼ ゲストユーザー機能

・ワンクリックゲストログイン

ゲストは以下を制限

・投稿やコメント書き込み不可

・プロフィール編集不可

・アカウント設定不可

▼ 管理者機能（Admin）

・管理者ログイン

・ユーザー一覧・詳細表示

・レビュー一覧表示（ID昇順）

・管理者専用レビュー詳細ページ

・レビュー削除機能（管理者権限）

🛠 今後追加予定の機能（任意）

・レスポンシブ対応（スマホ最適化）

・通報・非表示機能

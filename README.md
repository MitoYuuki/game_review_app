## アプリ名
game_review_app
## サイト概要
📝 サイトテーマ

ゲームの評価や感想を書き、様々な人に見てもらうレビューサイト。

💡 テーマを選んだ理由

私は小さいころからゲームに親しみ、これまでさまざまなタイトルを遊んできた経験があります。
しかし、ゲームの魅力や実際のプレイ感を知るための情報は散在しており、初心者や興味を持った人が作品を選びづらいという課題があります。

また、ユーザー目線で分かりやすくまとめられたレビューを求めている人も多く、信頼できる感想や評価へのニーズは高いと感じています。

こうした課題を解決するため、ゲームの魅力を整理して伝えられるレビューサイトを制作しました。

🎯 ターゲットユーザ

・対象のゲームを買うのに迷っている人

・プレイ時間の目安や満足度を知りたい人

・自分に合ったジャンルのゲームを探したい人

⏰ 主な利用シーン

・ゲームの難易度を知りたい時

・ゲームの簡単な物語の内容を知りたい時

・ゲームの想定プレイ時間を知りたい時

✨ 実装済み機能一覧
■レビュー機能
▼ ユーザー関連

・ユーザー登録 / ログイン（Devise）

・プロフィール編集

・マイページ表示

▼ 投稿（レビュー）機能

・レビュー投稿

・レビュー編集 / 削除（本人のみ）

・コメント機能

・いいね機能

・タグ付け

・ジャンル（グループ）分類

・レビュー評価付け

▼ 一覧表示・検索

・新着投稿一覧

・タグ別一覧

・ジャンル別一覧

・キーワード検索

・ユーザー検索

・投稿並び替え機能

▼ ゲストユーザー機能

・ワンクリックゲストログイン

ゲストは以下を制限

・投稿やコメント書き込み不可

・プロフィール編集不可

・アカウント設定不可

・フォロー操作不可

▼ 管理者機能（Admin）

・管理者ログイン

・ユーザー一覧・詳細表示

・レビュー一覧表示（ID昇順）

・管理者専用レビュー詳細ページ

・レビュー削除機能（管理者権限）

・コメント削除機能（管理者権限）

・ユーザー削除機能（管理者権限）

▼ レスポンシブ対応・UI改善

・トップページ、投稿一覧、投稿詳細、ユーザー関連ページのレスポンシブ対応

・スマホ表示でのレイアウト調整（サイドバー折りたたみ、カードレイアウト最適化）

■ コミュニティ機能（新規追加）

・コミュニティ作成 / 編集 / 削除

・コミュニティ参加 / 退会機能

・コミュニティ詳細ページの表示

・トピック作成 / 編集 / 削除

・コメント作成 / 編集 / 削除

・コミュニティオーナーは退会不可とし、オーナー不在状態を防ぐ設計にしています。

▼ トピック機能

・コミュニティ内でのトピック作成

・トピック編集 / 削除

・トピック一覧表示

・トピック詳細ページの表示

▼ トピックコメント機能

・コメント投稿

・コメント編集 / 削除

・コメント一覧表示

▼ 権限管理

・トピック/コメントの編集削除権限  
　└ 投稿者本人  
　└ コミュニティオーナー  
　└ サイト管理者（Admin）  

▼ 管理者機能の拡張（Admin）

・コミュニティ一覧表示

・コミュニティ詳細表示

・コミュニティ削除

・コミュニティ内トピック一覧表示

・トピック詳細表示

・トピック削除

・トピックコメント削除

■ バリデーション・エラーハンドリング

・日本語エラーメッセージ対応（I18n）

・エラーメッセージ重複表示の防止

・画面別に最適化したエラー表示

・Devise（登録・編集）エラー文言のカスタマイズ
